How do factors such as Crime, Income and Weather determine an area's population?

By MacBrandon Numfor Nden, Davin Wambogo, Sai Sreeramani, Prasanna Raj

Introduction

This project analyzes public data to help answer that question using three key metrics: income, crime, and weather. By curating, cleaning, and visualizing real-world datasets, and using machine learning (KMeans clustering), we aim to group regions by quality-of-life metrics and identify optimal districts for a user with preferences such as high income, low crime, and warm weather. This can help individuals gain insight regarding the best places to live.
